About

We are looking for a creative UI/UX Designer to design seamless, user-friendly experiences for web applications. You should have experience in wireframing, prototyping, and visual design using tools like Figma, Adobe XD, or Sketch. A basic understanding of user research, accessibility, and responsive design principles is required.

Responsibilities

  • Create wireframes, prototypes, and UI designs for web applications.
  • Conduct user research to improve usability.
  • Collaborate with developers to implement modern and responsive UI/UX.
  • Ensure consistency and accessibility across all designs.

Skills & Experience (0-1 Year)

  • Proficiency in Figma, Adobe XD, or Sketch.
  • Basic understanding of UI/UX principles & web design.
  • Ability to translate user needs into design solutions.
  • AKnowledge of HTML/CSS is a plus.

What We Offer

  • Opportunity to work on real-world projects.
  • Exposure to modern design tools & trends.
  • On-site work with a creative and supportive team.
  • On-site work in a collaborative and growth-focused environment.
